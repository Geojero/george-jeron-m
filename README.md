# george-jeron-m# George Jeron M Portfolio

A pixel-perfect replica of [venkateshrapolu.vercel.app](https://venkateshrapolu.vercel.app/)

## Tech Stack

- **Vite** + **React**
- **Tailwind CSS v4**
- **GSAP** (GreenSock Animation Platform) + ScrollTrigger
- **Space Grotesk** + **Unbounded** fonts from Google Fonts

## Features

- 🎯 Custom cursor with dot + trailing ring
- ⏳ Animated loading screen with progress bar
- 🌙 Dark/Light theme toggle
- 📱 Fully responsive (mobile-first)
- ✨ GSAP scroll animations (fade-up, stagger)
- 🃏 3D tilt effect on project cards
- 🎨 Neon lime green (#CEF441) accent

## Getting Started

```bash
npm install
npm run dev
```

## Sections

1. **Hero** — Big headline with outline text + CTA buttons
2. **About** — Bio, avatar, info cards
3. **Skills** — Categorized tech stack chips
4. **Projects** — 4 featured projects with 3D hover
5. **Coding Profiles** — LeetCode, GitHub, SkillRack stats
6. **Contact** — Form + social links
7. **Footer**
